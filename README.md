# Сравнение методов решения СЛАУ — Python

Реализованы метод Гаусса без выбора ведущего элемента, метод Гаусса с частичным выбором главного элемента, LU-разложение без перестановок, а также прямая и обратная подстановки.

## Что делает программа

Программа запускает 3 серии экспериментов:

1. Сравнение времени решения одной системы для размеров `n = 100, 200, 400, 600`.
2. Сравнение времени при нескольких правых частях для `n = 200` и `k = 1, 5, 20`.
3. Проверка точности на матрицах Гильберта размеров `n = 5, 10, 15`.

Результаты сохраняются в CSV и PNG.

## Зависимости

```bash
pip install numpy matplotlib
```

## Запуск

```bash
python main.py
```

## Структура

- `main.py` — весь код лабораторной;
- `results_one_system.csv` — таблица времени для одной системы;
- `results_many_rhs.csv` — таблица времени для нескольких правых частей;
- `results_hilbert.csv` — таблица погрешности и невязки;
- `plot_one_system.png`, `plot_many_rhs.png`, `plot_hilbert.png` — графики.

## Краткие выводы

- Для одной системы метод Гаусса быстрее LU-разложения.
- При нескольких правых частях LU становится выгоднее, потому что разложение считается один раз.
- На матрице Гильберта точность заметно ухудшается с ростом размера.
