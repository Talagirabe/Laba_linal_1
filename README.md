# Сравнение методов решения СЛАУ — Python

Реализованы метод Гаусса без выбора ведущего элемента, метод Гаусса с частичным выбором главного элемента, LU-разложение без перестановок, а также прямая и обратная подстановки.

## Что делает программа

Программа запускает 3 серии экспериментов:

1. Сравнение времени решения одной системы для размеров `n = 100, 200, 400, 600`.
2. Сравнение времени при нескольких правых частях для `n = 200` и `k = 1, 5, 20`.
3. Проверка точности на матрицах Гильберта размеров `n = 5, 10, 15`.

Результаты сохраняются в CSV и PNG.

## 1. Постановка задачи

Рассматривается система линейных уравнений:

Ax = b

Цель работы:
- реализовать методы решения
- сравнить их по времени и точности

Методы:
- Гаусс без выбора
- Гаусс с выбором
- LU-разложение

---

## 2. Математическое обоснование

Метод Гаусса:
приводит матрицу к верхнетреугольному виду.

После этого:

x_i = (b_i - сумма) / a_ii

---

Проблема:
если a_ii маленькое → ошибки растут.

---

Решение:
выбор главного элемента:

берём max |a_ik|

---

LU-разложение:

A = LU

Решение:
Ly = b  
Ux = y  

---

Преимущество:
LU делается один раз → потом быстро решаем много систем.

---

Матрица Гильберта:

H_ij = 1 / (i + j - 1)

Плохо обусловлена → тест устойчивости.
